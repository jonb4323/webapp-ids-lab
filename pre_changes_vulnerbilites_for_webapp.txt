Vulnerabilites for Web App (Known)

- Different user updatingg other user inforrmation by just knowing fields 
- Using get to login and the login credentials are sent in the url (Fix using POST)
- Being able to redirect based off the url (can jump straight to admin page)
- SQL Injection to pull users from database
